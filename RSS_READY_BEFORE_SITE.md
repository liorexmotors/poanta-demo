# Poanta RSS-only — מוכן לפני חיבור לאתר

## מה מוכן
- `rss_sources.json` — רשימת מקורות RSS תקינים בלבד.
- `scripts/prepare_rss_preview.py` — סקריפט הכנה בטוח שלא משנה את האתר.
- `rss_preview.json` — פלט JSON לבדיקה.
- `rss_preview.md` — דוח קריא של הרצה.
- `RSS_RESEARCH.md` — מחקר RSS: מומלץ/לא מומלץ.
- `RSS_APPROVAL_SAMPLE.md` — דוגמה להודעת אישור לפני פרסום.

## מה לא נעשה עדיין
- לא חובר ל־`feed.json`.
- לא חובר לעיצוב/אתר.
- לא נוצר פרסום אוטומטי.
- לא הופעל scraping/fallback.

## מקורות פעילים כרגע
- ynet חדשות / מבזקים / ספורט
- וואלה ראשי / חדשות / כלכלה
- ישראל היום
- הארץ / דעות / ספורט
- גלובס שוק ההון

## Gate לפני חיבור לאתר
לפני חיבור האתר צריך אישור מפורש מליאור:
1. לאשר RSS-only כמקור אמת.
2. לאשר את רשימת המקורות.
3. לאשר פורמט הודעת האישור.
4. לאשר האם הפרסום אחרי אישור נכנס ל־`feed.json` ידנית או דרך cron.

## המלצה מקצועית
להתחיל במשך שבוע במצב approval-first:
RSS → בחירת מועמדים → הודעת אישור בטלגרם → רק מאושרים נכנסים לפיד.
